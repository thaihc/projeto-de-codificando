(De)Codificando

Utilize esse site para criar seus códigos ou decodificar suas mensagens, utilizando Cifra de César ou Base64. A Cifra de César é um tipo de cifra de substituição, em que cada letra do texto é substituída por outra, de acordo com a quantidade de incrementos utilizados. Base64 é um método de codificação de dados para transferência na Internet, ele é utilizado para transmissão de dados binários que lidam com texto.
